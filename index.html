That Sky Events

This project was developed to support the Sky: Children of the Light community by providing a reliable, easy-to-access event calendar. It features a clean UI with a dynamic event card system and a calendar interface for quick navigation.

Built using HTML, CSS, and vanilla JavaScript, the site is hosted on Netlify and uses Firebase for storing dynamic content such as event data. ImageKit handles media delivery for optimal performance. The design emphasizes clarity, responsiveness, and accessibility especially for players using mobile devices.

Echoes of Light

Memory Game Web App

The Challenge

Most memory games feel generic and disconnected from their source of inspiration. As a fan and player of Sky: Children of the Light, I wanted to build an experience that felt immersive, visually thematic, and emotionally resonant for the community, something more than just a template-based game.

The Solution

A fully responsive, browser-based memory game built with React. Echoes of Light blends clean design with smooth animations, subtle audio cues, and a thematic motif that mirrors Sky’s ambient aesthetic. It's mobile-optimized and deployable with minimal load times.
